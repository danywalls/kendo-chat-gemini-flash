# Build a ChatLens with Kendo UI, Gemini and Angular 19

## Project Summary
Build a modern chatbot in Angular 19 using Google Gemini AI and Kendo UI for Angular's Conversational UI components.

## How to Run
### Prerequisites
* Google AI API Key

### Installation
1. `npm install` 

### Running
1. Configure your API key in the environment files.
2. `ng serve`
